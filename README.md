# LockIn

**LockIn** is a collaborative study platform built with **JavaFX**, designed to help students study collaboratively,
share resources, and stay productive through structured study sessions.

The application combines real-time communication, study room management, and academic resource sharing into a single
desktop application.

---

## Features

### Quality of Life Features

* **Login / Logout system**
* **Theme switching (Light/Dark)**
* **Profile modification**

    * Update personal information
    * Manage uploaded content

---

### Main Features

#### 1. Study Rooms

Collaborative virtual study environments.

* Group chat
* File sharing inside rooms
* Pomodoro study timers
* Voice chat *(planned)*
* Video chat *(planned)*

---

#### 2. Central Posting System

A shared public space for collaboration.

* Create public posts
* Find project partners

---

#### 3. Profile

Each user profile acts as a resource.

* Upload notes, solutions, and study materials
* Public or private visibility

---

## Tech Stack

* **Language:** Java (Temurin 21.0.9)
* **UI Framework:** JavaFX 21.0.6
* **Build Tool:** Gradle 8.13

---

## Getting Started

### Prerequisites

* Java JDK **(Temurin 21.0.9)**
* Gradle **(8.13)**
* JavaFX SDK **(21.0.6)**

---

### Clone the Repository

```bash
git clone https://github.com/kazimdraiyan/LockIn.git
cd LockIn
```

---

### Run the Project

```bash
./gradlew run
```

---

### Build the Project

```bash
./gradlew build
```
